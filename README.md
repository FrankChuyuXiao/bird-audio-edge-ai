## Dataset

Due to GitHub storage limitations, the full audio dataset is not included in this repository.

The dataset consists of five North American bird species:

- American Robin
- Blue Jay
- House Sparrow
- Mourning Dove
- Northern Cardinal

Bird recordings were collected from public bioacoustic resources such as Xeno-canto.

Users should place audio files in the following structure:

audio/
├── American_Robin/
├── Blue_Jay/
├── House_Sparrow/
├── Mourning_Dove/
└── Northern_Cardinal/
